## Mt. Kapigpiglatan + Veto Falls + Liwliwa Beach (December 28, 2024)

### Inclusions:
- Van rental RT
- Guide fee
- Registration fee
- Bagtags
- Drone Shots

### Exclusions:
- Shower
- Breakfast and Lunch, including trail foods
- Entrance resort to Liwliwa (Free Cottage)
- Others not mentioned above

- **10:00 PM** - ETA MOA
